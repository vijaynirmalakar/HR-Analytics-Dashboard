# HR Analytics Dashboard — Power BI

A Power BI dashboard built to analyze employee attrition.
Dataset: 1,416 employees | Attrition rate: 16.17%

## What's in the dashboard
- Attrition by Education, Age Group, Job Role
- Attrition by Salary Slab and Years at Company
- Gender split and Job Satisfaction matrix
- Department filter (HR / R&D / Sales)

## Key findings
- 67% of attrition is from employees earning below 5K/month
- Year 1 is the riskiest — 53 employees left in their first year
- Age group 26–35 has the highest number of exits (111)

## Tools used
Power BI Desktop | Power Query | DAX
